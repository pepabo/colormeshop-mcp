# Security Policy

Please report security vulnerabilities affecting this repository or the hosted Color Me Shop MCP server at `https://agent.colorme.app/api/mcp` through GMO Pepabo's official vulnerability reporting channel:

- https://pepabo.com/contact/vulnerability-reporting/ (Japanese)
- https://pepabo.com/en/contact/vulnerability-reporting/ (English)

Do not report security issues through public GitHub issues, discussions, or pull requests.
